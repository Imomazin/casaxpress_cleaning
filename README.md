# Strathmore Cleaning

Professional cleaning services website built with Next.js, TypeScript, and Tailwind CSS.

## Description

Strathmore Cleaning is a modern, responsive website for a professional cleaning services company. The site showcases residential, commercial, and deep cleaning services.

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **Vercel** - Deployment platform

## Getting Started in StackBlitz

This project is designed to run seamlessly in StackBlitz:

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Preview the App**
   - StackBlitz will automatically open a preview window
   - The app will be available at the local development URL
   - Changes will hot-reload automatically

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the production application
- `npm start` - Start the production server
- `npm run lint` - Run ESLint for code quality

## Deployment

This project is optimized for deployment on Vercel:

1. Push your code to GitHub
2. Import the repository in Vercel
3. Vercel will automatically detect Next.js and deploy
